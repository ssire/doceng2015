Exercice 10

Enoncé de l'exercice 10

En observant le contenu des données dans le fichier euro-foot.xml, imaginez une interface REST pour accéder aux données. Cette interface devrait offrir les services suivants :

  • la liste des groupes
  • la liste des pays par groupe
  • les statisiques de qualification pour chaque pays
  • la composition de l'équipe de chaque pays
  • les renseignements sur chaque joueur
  • la liste des joueurs ayant une taille comprise entre un minimum et un maximum
  • la liste des joueurs ayant un poids compris entre un minimum et un maximum

Contraintes à prendre en compte

La base de données contient les informations pour l'euro-foot 2008, pour l'exercice imaginez que l'on puisse utiliser l'application sur toutes les éditions de l'euro-foot. Vous devez également prévoir les mises à jour pour construire les données (ajouter un nouvel euro-foot, ajouter un groupe, ajouter un pays dans un groupe, etc. et les mises à jour corrspondantes).
